# Text Component

The Text component provides a consistent way to display text content across the application with proper typography, styling options, and accessibility features.

## Features

- **Multiple variants**: body, paragraph, caption, label, code, quote
- **Semantic HTML**: Renders as appropriate HTML element based on variant or custom element
- **Rich styling options**: size, weight, alignment, color, and more
- **Responsive sizing**: Adjust size at different breakpoints
- **Text utilities**: truncation, transformation, decoration
- **Accessibility**: Screen reader support and proper semantic structure
- **Dark Mode Support**: Automatically adapts to light/dark themes

## Usage Examples

### Basic Usage

```tsx
import { Text } from "@nas-net/core-ui-qwik";

export default component$(() => <Text>This is a standard body text.</Text>);
```

### Variants

```tsx
<Text variant="body">Standard body text</Text>
<Text variant="paragraph">Paragraph text with block styling</Text>
<Text variant="caption">Smaller caption text</Text>
<Text variant="label">Label text (often used with form fields)</Text>
<Text variant="code">Code text with monospace font</Text>
<Text variant="quote">Blockquote with special styling</Text>
```

### Text Sizes

```tsx
<Text size="xs">Extra small text</Text>
<Text size="sm">Small text</Text>
<Text size="base">Base size text (default)</Text>
<Text size="lg">Large text</Text>
<Text size="xl">Extra large text</Text>
<Text size="2xl">Double extra large text</Text>
```

### Font Weights

```tsx
<Text weight="light">Light weight text</Text>
<Text weight="normal">Normal weight text (default)</Text>
<Text weight="medium">Medium weight text</Text>
<Text weight="semibold">Semibold weight text</Text>
<Text weight="bold">Bold weight text</Text>
<Text weight="extrabold">Extra bold weight text</Text>
```

### Text Alignment

```tsx
<Text align="left">Left aligned text (default)</Text>
<Text align="center">Center aligned text</Text>
<Text align="right">Right aligned text</Text>
```

### Color Variants

```tsx
<Text color="primary">Primary text color (default)</Text>
<Text color="secondary">Secondary text color</Text>
<Text color="tertiary">Tertiary text color</Text>
<Text color="subtle">Subtle text color</Text>
<Text color="accent">Accent color for emphasis</Text>
<Text color="success">Success message color</Text>
<Text color="warning">Warning message color</Text>
<Text color="error">Error message color</Text>
<Text color="info">Informational message color</Text>

{/* For dark backgrounds */}
<div class="bg-gray-900 p-4">
  <Text color="inverse">Inverse text color</Text>
</div>
```

### Text Utilities

```tsx
{/* Text transformation */}
<Text transform="uppercase">uppercase text</Text>
<Text transform="lowercase">LOWERCASE TEXT</Text>
<Text transform="capitalize">capitalize first letter</Text>

{/* Text decoration */}
<Text decoration="underline">Underlined text</Text>
<Text decoration="line-through">Strikethrough text</Text>

{/* Font styling */}
<Text italic>Italic text</Text>
<Text monospace>Monospace font text</Text>

{/* Truncation */}
<div class="w-48">
  <Text truncate>This long text will be truncated with an ellipsis</Text>
</div>

{/* Multi-line truncation */}
<div class="w-48">
  <Text truncate maxLines={2}>
    This long text will be truncated after two lines with an ellipsis
    to indicate that there is more content available.
  </Text>
</div>
```

### Responsive Text

```tsx
<Text
  responsiveSize={{
    base: "sm", // Small on mobile
    md: "base", // Base size on tablets (768px+)
    lg: "lg", // Large on desktops (1024px+)
    xl: "xl", // Extra large on large screens (1280px+)
  }}
>
  This text will change size at different breakpoints
</Text>
```

### Accessibility Support

```tsx
{
  /* Screen reader only text (visually hidden) */
}
<Text srOnly>This text is only announced to screen readers</Text>;

{
  /* Descriptive text for an icon button */
}
<button aria-label="Close">
  <Icon name="x" />
  <Text srOnly>Close dialog</Text>
</button>;
```

### Interactive Text

```tsx
<Text
  color="accent"
  decoration="underline"
  onClick$={() => console.log("Text clicked")}
>
  This text is clickable like a link
</Text>
```

## Accessibility Considerations

- Use semantic HTML elements appropriate for your content
- Avoid using color as the only means of conveying information
- Ensure sufficient color contrast ratios for all color variants
- Consider using `srOnly` to provide additional context for screen readers
- When making text interactive with `onClick$`, ensure it has appropriate focus styling and keyboard support

## Design Tokens

This component uses the following design tokens:

- Text colors: `text-gray-900`, `text-gray-700`, etc.
- Dark mode text colors: `dark:text-gray-100`, `dark:text-gray-300`, etc.
- Font weights from Tailwind's font weight scale
- Font sizes from Tailwind's size scale
