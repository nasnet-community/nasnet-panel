# Link Component

The Link component provides a consistent way to display links across the application, with seamless integration for both internal navigation (Qwik routing) and external links.

## Features

- **Automatic Link Type Detection**: Automatically determines internal vs external links
- **Qwik City Integration**: Uses Qwik's Link component for client-side routing
- **Multiple Variants**: Standard, button, navigation, subtle, icon, and breadcrumb styles
- **Rich Styling Options**: Size, weight, color, and underline styles
- **Icon Support**: Optional prefix and suffix icons
- **Security Features**: Automatic `rel="noopener noreferrer"` for external links
- **Accessibility Support**: Proper ARIA attributes and keyboard interaction
- **Dark Mode Support**: Automatically adapts to light/dark themes

## Usage Examples

### Basic Usage

```tsx
import { Link } from "@nas-net/core-ui-qwik";

export default component$(() => (
  <>
    {/* Internal link (uses Qwik routing) */}
    <Link href="/about">About Us</Link>

    {/* External link */}
    <Link href="https://qwik.builder.io/">Qwik Documentation</Link>
  </>
));
```

### Link Variants

```tsx
{
  /* Standard link (default) */
}
<Link href="/standard">Standard Link</Link>;

{
  /* Button-style link */
}
<Link href="/button" variant="button">
  Button Link
</Link>;

{
  /* Navigation link */
}
<Link href="/nav" variant="nav">
  Nav Link
</Link>;

{
  /* Subtle link */
}
<Link href="/subtle" variant="subtle">
  Subtle Link
</Link>;

{
  /* Link with icons */
}
<Link href="/icons" variant="icon" prefixIcon={<Icon name="link" />}>
  Icon Link
</Link>;

{
  /* Breadcrumb link */
}
<Link href="/breadcrumb" variant="breadcrumb">
  Breadcrumb Link
</Link>;
```

### External Links

External links (URLs starting with http/https) are automatically handled with proper security attributes:

```tsx
{
  /* Opens in new tab with rel="noopener noreferrer" */
}
<Link href="https://example.com">External Link</Link>;

{
  /* With custom icon */
}
<Link href="https://example.com" suffixIcon={<ExternalLinkIcon />}>
  External Link with Icon
</Link>;

{
  /* Force link to be treated as external */
}
<Link href="/api/data" external>
  Forced External Link
</Link>;
```

### With Icons

```tsx

{
  /* Suffix icon */

  prefixIcon={<Icon name="link" />}

  Link with Icons

```

### State Variants

```tsx
{
  /* Disabled link */
}

  Disabled Link
</Link>;

{
  /* Active link (for navigation) */
}
<Link href="#" active>
  Active Link
</Link>;

{
  /* Active navigation link */

<div class="flex border-b">
  <Link href="/home" variant="nav">
    Home
  </Link>
  <Link href="/features" variant="nav">
    Features
  </Link>
  <Link href="/products" variant="nav" active>

  </Link>
```

### Special Use Cases

```tsx
{
  /* Truncated link for long text */
}
<div class="w-40">
  <Link href="#" truncate>
    This is a very long link that will be truncated with ellipsis
  </Link>
</div>;

{
  /* Link with click handler */
}
<Link
  href="#"
  onClick$={(e) => {
    e.preventDefault();
    console.log("Link clicked!");
  }}
>
  Link with Click Handler
</Link>;

{
  /* Link with custom props for Qwik's Link component */
}
<Link href="/products/123" qwikCity={{ reload: true }}>
  Reload Product Page
</Link>;
```

## Props Reference

| Prop         | Type                                                                                                                          | Default      | Description                               |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------- | ------------ | ----------------------------------------- |
| `href`       | `string`                                                                                                                      | (required)   | Link destination URL or path              |
| `external`   | `boolean`                                                                                                                     | Auto-detect  | Whether to treat as external link         |
| `variant`    | `"standard" \| "button" \| "nav" \| "subtle" \| "icon" \| "breadcrumb"`                                                       | `"standard"` | Link variant style                        |
| `size`       | `"xs" \| "sm" \| "base" \| "lg" \| "xl"`                                                                                      | `"base"`     | Font size                                 |
| `weight`     | `"normal" \| "medium" \| "semibold" \| "bold"`                                                                                | `"medium"`   | Font weight                               |
| `color`      | `"primary" \| "secondary" \| "tertiary" \| "inverse" \| "accent" \| "inherit" \| "success" \| "error" \| "warning" \| "info"` | `"primary"`  | Link color                                |
| `underline`  | `"none" \| "hover" \| "always" \| "animate"`                                                                                  | `"hover"`    | Underline style                           |
| `newTab`     | `boolean`                                                                                                                     | Auto-detect  | Open in new tab                           |
| `prefixIcon` | `JSXNode`                                                                                                                     | -            | Icon before link text                     |
| `suffixIcon` | `JSXNode`                                                                                                                     | -            | Icon after link text                      |
| `truncate`   | `boolean`                                                                                                                     | `false`      | Truncate text with ellipsis               |
| `disabled`   | `boolean`                                                                                                                     | `false`      | Disable the link                          |
| `active`     | `boolean`                                                                                                                     | `false`      | Mark as active (for navigation)           |
| `secure`     | `boolean`                                                                                                                     | `true`       | Add security attributes to external links |
| `rel`        | `string`                                                                                                                      | -            | Custom rel attribute                      |
| `ariaLabel`  | `string`                                                                                                                      | -            | Accessibility label                       |
| `id`         | `string`                                                                                                                      | -            | ID attribute                              |
| `class`      | `string`                                                                                                                      | -            | Additional CSS classes                    |
| `children`   | `string \| JSXNode`                                                                                                           | -            | Link content                              |
| `onClick$`   | `PropFunction<(e: MouseEvent) => void> \| QRL<(e: MouseEvent) => void>`                                                       | -            | Click handler                             |
| `qwikCity`   | `Partial<QwikLinkProps>`                                                                                                      | -            | Props for Qwik's Link component           |

## Accessibility Considerations

- Always provide descriptive link text that clearly indicates the destination
- For links with icon only, use the `ariaLabel` prop to provide an accessible name
- Use the `active` prop for navigation links to indicate the current page
- For links that open in new tabs, consider including visual indicators
- Ensure color contrast meets WCAG guidelines (especially for subtle links)
- The component automatically adds `rel="noopener noreferrer"` to external links that open in new tabs

## Design Tokens

This component uses the following design tokens:

- Text colors: `text-blue-600`, `text-gray-700`, etc.
- Dark mode text colors: `dark:text-blue-400`, `dark:text-gray-300`, etc.
- Focus styles: `focus:ring-2`, `focus:ring-blue-500`, etc.
- Border colors: `border-current`, `border-gray-200`, etc.
- Font weights from Tailwind's font weight scale
