# Box Component

The Box component is a versatile layout container that serves as a fundamental building block in the Connect design system. It provides a consistent way to create containers with various styling options like padding, margin, borders, background colors, and shadows.

## Usage

```tsx
import { Box } from "@/components/Core/Layout/Box";

export default component$(() => {
  return (
    <Box padding="md" backgroundColor="surface" borderRadius="md">
      Content goes here
    </Box>
  );
});
```

## Props

### Basic Props

| Prop       | Type                          | Default | Description                                |
| ---------- | ----------------------------- | ------- | ------------------------------------------ |
| `children` | `JSXNode`                     | -       | Content to be rendered inside the box      |
| `as`       | `keyof QwikIntrinsicElements` | `'div'` | HTML tag to be used for the box            |
| `class`    | `string`                      | -       | Additional class names to apply to the box |

### Spacing Props

| Prop      | Type                                                                                                                                                | Default  | Description        |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ------------------ |
| `padding` | `BoxPadding \| { all?: BoxPadding; x?: BoxPadding; y?: BoxPadding; top?: BoxPadding; right?: BoxPadding; bottom?: BoxPadding; left?: BoxPadding; }` | `'none'` | Padding of the box |
| `margin`  | `BoxMargin \| { all?: BoxMargin; x?: BoxMargin; y?: BoxMargin; top?: BoxMargin; right?: BoxMargin; bottom?: BoxMargin; left?: BoxMargin; }`         | `'none'` | Margin of the box  |

### Border Props

| Prop           | Type                                                                                              | Default     | Description              |
| -------------- | ------------------------------------------------------------------------------------------------- | ----------- | ------------------------ |
| `borderRadius` | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'`                                        | `'none'`    | Border radius of the box |
| `borderWidth`  | `'none' \| 'thin' \| 'normal' \| 'thick'`                                                         | `'none'`    | Border width of the box  |
| `borderStyle`  | `'solid' \| 'dashed' \| 'dotted' \| 'double' \| 'none'`                                           | `'solid'`   | Border style of the box  |
| `borderColor`  | `'default' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| 'info' \| 'muted'` | `'default'` | Border color of the box  |

### Visual Props

| Prop              | Type                                                                                                                                | Default         | Description                 |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------- | --------------- | --------------------------- |
| `backgroundColor` | `'transparent' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| 'info' \| 'muted' \| 'surface' \| 'surface-alt'` | `'transparent'` | Background color of the box |
| `shadow`          | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| 'inner'`                                                                        | `'none'`        | Shadow of the box           |

### Layout Props

| Prop         | Type      | Default | Description                                                       |
| ------------ | --------- | ------- | ----------------------------------------------------------------- |
| `fullWidth`  | `boolean` | `false` | Set to true to make the box take the full width of its container  |
| `fullHeight` | `boolean` | `false` | Set to true to make the box take the full height of its container |

### Accessibility Props

| Prop               | Type     | Default | Description                  |
| ------------------ | -------- | ------- | ---------------------------- |
| `role`             | `string` | -       | ARIA role for the box        |
| `aria-label`       | `string` | -       | ARIA label for the box       |
| `aria-labelledby`  | `string` | -       | ARIA labelledby for the box  |
| `aria-describedby` | `string` | -       | ARIA describedby for the box |

## Examples

### Basic Box with Padding and Border Radius

```tsx
<Box padding="md" backgroundColor="surface" borderRadius="md">
  Basic Box with medium padding and rounded corners
</Box>
```

### Box with Custom Border

```tsx
<Box
  padding="md"
  backgroundColor="surface"
  borderRadius="md"
  borderWidth="normal"
  borderStyle="dashed"
  borderColor="primary"
>
  Box with a dashed primary-colored border
</Box>
```

### Box with Shadow

```tsx
<Box padding="md" backgroundColor="surface" borderRadius="md" shadow="lg">
  Box with a large shadow
</Box>
```

### Box with Directional Padding

```tsx
<Box
  backgroundColor="surface"
  borderRadius="md"
  padding={{
    x: "lg", // Horizontal padding
    y: "sm", // Vertical padding
  }}
>
  Box with different horizontal and vertical padding
</Box>
```

### Box as a Different HTML Element

```tsx
<Box as="section" padding="md" backgroundColor="surface" borderRadius="md">
  This is a section element
</Box>
```

### Box with Accessibility Attributes

```tsx
<Box
  padding="md"
  backgroundColor="surface"
  borderRadius="md"
  role="region"
  aria-label="Important information"
>
  Box with ARIA attributes for accessibility
</Box>
```

## Padding and Margin Size Reference

| Size   | Value  | Description                   |
| ------ | ------ | ----------------------------- |
| `none` | `0px`  | No padding/margin             |
| `xs`   | `4px`  | Extra small padding/margin    |
| `sm`   | `8px`  | Small padding/margin          |
| `md`   | `16px` | Medium padding/margin         |
| `lg`   | `24px` | Large padding/margin          |
| `xl`   | `32px` | Extra large padding/margin    |
| `2xl`  | `40px` | 2x extra large padding/margin |
| `3xl`  | `48px` | 3x extra large padding/margin |

## Border Radius Reference

| Size   | Description         |
| ------ | ------------------- |
| `none` | No border radius    |
| `xs`   | 2px border radius   |
| `sm`   | 4px border radius   |
| `md`   | 6px border radius   |
| `lg`   | 8px border radius   |
| `xl`   | 12px border radius  |
| `full` | Fully rounded (50%) |

## Border Width Reference

| Size     | Description |
| -------- | ----------- |
| `none`   | No border   |
| `thin`   | 1px border  |
| `normal` | 2px border  |
| `thick`  | 4px border  |

## Accessibility

The Box component supports various ARIA attributes to improve accessibility:

- `role`: Defines the role of the element in the accessibility tree
- `aria-label`: Provides a text label
- `aria-labelledby`: References another element's ID that labels this element
- `aria-describedby`: References another element's ID that describes this element

## Dark Mode Support

The Box component automatically adapts to dark mode through Tailwind CSS classes. Default styles change appropriately when the user's system is in dark mode or when the site-wide dark mode is enabled.
