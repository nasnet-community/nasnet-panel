# Divider

The Divider component is a layout primitive that creates a visual separation between content sections.

## Overview

Divider is a flexible layout component that provides a visual separation with options for:

- Horizontal or vertical orientation
- Different thickness variants
- Style variants (solid, dashed, dotted)
- Color options
- Optional labeled content
- Customizable spacing

## Usage

```tsx
import { Divider } from '@connect/Core/Layout/Divider';

// Basic horizontal divider
<Divider />

// Vertical divider
<Divider orientation="vertical" />

// Styled divider
<Divider thickness="medium" variant="dashed" color="primary" />

// Divider with label
<Divider label="OR" labelPosition="center" />
```

## Accessibility

The Divider component includes proper ARIA attributes:

- `role="separator"` is applied by default
- `aria-orientation` matches the visual orientation
