# Stack

The Stack component is a layout primitive that makes it easy to arrange items in vertical or horizontal stacks with consistent spacing.

## Overview

Stack is a versatile layout component that provides a simple way to:

- Arrange elements in a column or row
- Apply consistent spacing between elements
- Control alignment and justification
- Support responsive layouts
- Add optional dividers between elements
- Handle RTL layouts

## Usage

```tsx
import { Stack } from '@connect/Core/Layout/Stack';

// Basic vertical stack (default)
<Stack spacing="md">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>

// Horizontal stack
<Stack direction="row" spacing="md">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>

// Responsive stack (column on mobile, row on larger screens)
<Stack
  direction={{
    base: 'column',
    md: 'row'
  }}
  spacing="md"
>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>
```

### Spacing Options

<Story of={StackStories.SpacingExamples} />

The Stack component supports various spacing options: `none`, `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, and `3xl`.

### Alignment and Justification

<Story of={StackStories.AlignmentExamples} />

Control how items are positioned along the main and cross axes:

- `justify`: Controls alignment along the main axis

  - Options: `start`, `center`, `end`, `between`, `around`, `evenly`

- `align`: Controls alignment along the cross axis
  - Options: `start`, `center`, `end`, `stretch`, `baseline`

### Wrapping

<Story of={StackStories.WrappingStack} />

When space is limited, items can wrap to the next line:

```tsx
<Stack direction="row" wrap="wrap" spacing="md">
  <div>Item 1</div>
  <div>Item 2</div>
  ...
</Stack>
```

### Dividers

<Story of={StackStories.WithDividers} />

Add dividers between stack items with the `dividers` prop:

```tsx
<Stack dividers dividerColor="muted">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>
```

### Reverse Order

<Story of={StackStories.ReverseOrder} />

Reverse the display order of items with the `reverse` prop:

```tsx
<Stack direction="row" reverse>
  <div>Item 1</div> {/* Will appear last */}
  <div>Item 2</div>
  <div>Item 3</div> {/* Will appear first */}
</Stack>
```

### Nested Stacks

<Story of={StackStories.NestedStacks} />

Stacks can be nested to create complex layouts:

```tsx
<Stack direction="column" spacing="lg">
  <div>Header</div>

  <Stack direction="row" spacing="md">
    <div>Sidebar</div>
    <div>
      <Stack direction="column">
        <div>Content</div>
        <div>More Content</div>
      </Stack>
    </div>
  </Stack>

  <div>Footer</div>
</Stack>
```

## Accessibility

The Stack component includes several features to ensure accessibility:

- Semantic HTML structure with appropriate ARIA attributes
- Support for RTL languages through the `supportRtl` prop
- Customizable roles via the `role` prop and labels via `aria-label`

## Best Practices

1. **Choose the right direction**: Use vertical stacks for lists and sections that should appear in sequence. Use horizontal stacks for related elements that should appear side-by-side.

2. **Be consistent with spacing**: Use the same spacing value for similar UI patterns throughout your application.

3. **Use responsive direction**: Consider how your layout should adapt to different screen sizes by using the responsive direction feature.

4. **Avoid overly nested stacks**: While nesting is powerful, excessive nesting can lead to complex layouts that are difficult to maintain.

5. **Be mindful of RTL support**: If your application supports RTL languages, ensure the `supportRtl` prop is set to `true` (default).
