# Spacer

The Spacer component is a layout primitive that creates consistent whitespace between UI elements.

## Overview

Spacer is a flexible layout component that provides:

- Consistent vertical or horizontal spacing
- Responsive sizing options
- Flexible spacing that expands to fill available space
- Mobile-specific behavior (hide on small screens)

## Usage

```tsx
import { Spacer } from '@connect/Core/Layout/Spacer';

// Basic vertical spacer (default)
<div>Content above</div>
<Spacer size="md" />
<div>Content below</div>

// Horizontal spacer
<div class="flex">
  <div>Left content</div>
  <Spacer horizontal size="md" />
  <div>Right content</div>
</div>

// Flexible spacer (pushes content apart)
<div class="flex w-full">
  <div>Left aligned</div>
  <Spacer horizontal isFlexible />
  <div>Right aligned</div>
</div>

// Responsive spacer
<Spacer
  size={{
    base: 'sm',  // Small on mobile
    md: 'lg'     // Large on medium screens and up
  }}
/>
```

- The Spacer component has `aria-hidden="true"` since it's purely presentational
- It uses `role="none"` to indicate it has no semantic role
